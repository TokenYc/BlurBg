# BlurBg
一个用于对view的背景进行高斯模糊化处理的工具

```
BlurBg.blur(
            BlurConfig.into(imv_center)
                .setBgCorner(50.0f)
                .setNormalCoverColor(Color.parseColor("#22000000"))
                .setPressedCoverColor(Color.parseColor("#99000000"))
                .build()
        )
```
